What We Provide

The CDNGI Geoportal provides users with easy access to a wide range of spatial datasets and related resources focused on Cape Town. Users can explore data through an interactive map tool and access detailed dataset information in the metadata catalogue. The portal integrates web mapping services (WMS) and web feature services (WFS) which are accessible within the metadata catalogue, along with options for direct file downloads of certain datasets.
