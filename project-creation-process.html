About CDNGI Project

The Chief Directorate: National Geo-spatial Information (CDNGI) is South Africa's national mapping agency responsible for geodetic control, topographic mapping, and geospatial data management. The CDNGI Geoportal is a prototype platform for accessing spatial datasets tailored for the Cape Town area.

Project Creation Process

The development of the CDNGI Geoportal followed a structured process designed to implement a fully functional Spatial Data Infrastructure (SDI) for Cape Town. The process ensured that spatial datasets were not only accessible, but also usable, interoperable, and standards-compliant.

The result is a modern, open-access Geoportal supporting spatial data discovery, visualization, and download—custom-built for the case study area and fully aligned with SDI principles.

What We Provide

  • Access to geodetic control and cadastral datasets
  • Downloadable shapefiles and metadata
  • Interactive mapping via Leaflet with dynamic layers
  • WMS/WFS services hosted via GeoServer
  • Fully documented and standards-compliant metadata
