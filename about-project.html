About the CDNGI Geoportal Project

This geoportal was developed as part of a Spatial Data Infrastructures (SDI) course project in response to a brief from a hypothetical large organization with geographically distributed offices. The organization required a robust SDI to improve spatial data storage, sharing, and processing. Using the Chief Directorate: National Geo-spatial Information (CDNGI) as the case study and Cape Town as the study area, the project involved designing and implementing a new SDI framework. This included establishing a centralized geo-database, creating fictitious spatial datasets to simulate realistic organizational needs, and developing essential SDI components such as metadata, web mapping services (WMS), and web feature services (WFS). The result is a fully functional geoportal that demonstrates how spatial infrastructures can be implemented to support distributed users across an organization.
