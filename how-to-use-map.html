How to Use the Interactive Map

The Interactive Map allows users to visually explore spatial datasets hosted on the CDNGI Geoportal. This includes aerial imagery, topographic maps, elevation models, cadastral data, and more. Each dataset is organized by geographic grid tiles covering South Africa.

To effectively use the map, follow the steps below to familiarize yourself with all its features and functionality:

  1. Searching by Suburb Name

    To search for spatial data in a specific suburb:

    • Click the magnifying glass icon in the top-right corner of the map interface.
    • A search bar will appear. Start typing the name of the suburb you are interested in (e.g., "Claremont").
    • Select the matching result from the dropdown list.
    • The map will automatically zoom into the selected suburb, allowing you to explore the datasets available in that area.

    This is useful for locating spatial data in a known suburb without manually navigating the map.

  2. Searching by Grid Number

    To locate data using a specific map grid cell number:

    • Click the grid search icon (typically represented by a numbered square symbol).
    • Enter the grid cell number (e.g., "B14") into the input box.
    • The map will highlight and zoom to the corresponding grid cell.

    This is helpful if you already know your area of interest by grid reference, or if using referenced metadata.

  3. Viewing Grid Numbers on Hover

    Each grid cell has a visible number displayed in a popup when hovered over or clicked:

    • Move your mouse over a grid cell to see its grid cell number appear.
    • Use this number to search the grid directly or match with metadata entries.

    This is useful for identifying and referencing areas when downloading datasets.

  4. Accessing Metadata Through Grid Clicking

    Each grid cell on the map links directly to relevant spatial datasets:

    • Click on any grid cell on the map.
    • A side pane will slide open from the right.
    • This pane displays dataset categories available in the selected grid, such as:
      • Aerial Imagery
      • Elevation Data
      • Beacons
      • Lease Areas
      • Farms
      • Township Allotments
      • Streets
    • Click on any category in the list to be taken to the corresponding dataset’s metadata entry in the catalogue.
    • From the catalogue, you can:
      • View detailed metadata (title, source, CRS, abstract, credits)
      • Preview the dataset
      • Download available files

    This integration allows easy access to relevant datasets directly from the map without browsing the full catalogue.

  5. Using the Geolocation Tool

    Surveyors and field users can take advantage of the geolocation feature:

    • Click the target icon (typically a dot with surrounding rings).
    • If prompted, allow location access in your browser.
    • The map will zoom to your current location and place a marker on the map.

    This feature is especially useful for:

    • Field surveyors identifying available datasets nearby
    • Mobile users needing quick access to data while on-site

  6. Toggling Layers with the Layer Control Panel

    You have full control over which spatial datasets are visible on the map:

    • Open the Layer Control Panel, usually found on the top-left corner or as a collapsible sidebar.
    • A list of all available map layers will be shown (e.g., Roads, Beacons, Topography, DEM).
    • Check or uncheck boxes to toggle visibility of specific layers.

    This feature allows you to customize your viewing experience and focus only on relevant datasets.

  7. Navigating Back to the Homepage

    Click the CDNGI logo at the top of the map page at any time to:

    • Return to the geoportal homepage
    • Access other components such as the metadata catalogue or project background

    This provides a quick way to return to the main site without losing your place in the map.

  8. Using Zoom Controls

    Zooming can be done using the plus (+) and minus (-) icons located in the top-right corner of the map.

    • Click the plus icon (+) to zoom in
    • Click the minus icon (-) to zoom out

    If you’re having difficulty viewing grid cell details or map layers, these controls allow you to adjust your view easily.

Tips for Best Use

  • Use the grid number popups to note and reuse your areas of interest.
  • Use the geolocation tool in the field to discover nearby datasets.
  • If a dataset appears to be missing, check the layer panel to ensure the layer is turned on.
  • Always allow browser pop-ups if you’re having trouble accessing metadata links from the map.
